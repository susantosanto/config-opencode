# AwesomeDesign-md - Build 100X Stunning Designs

## 📋 Menu Perintah

| Perintah | Aksi |
|---------|------|
| `/awesome-design` | Tampilkan menu utama |
| `install [brand]` | Install design brand tertentu |
| `list` | Lihat semua design yang tersedia |
| `preview [brand]` | Preview design di browser |

---

## 🚀 Quick Start

### Opsi 1: Clone Repo
```bash
git clone https://github.com/VoltAgent/awesome-design-md
cp design-md/stripe.ai/DESIGN.md ./DESIGN.md
```

### Opsi 2: Fetch Langsung
```bash
npx degit VoltAgent/awesome-design-md/design-md/stripe.ai DESIGN.md
```

### Opsi 3: Via npx getdesign
```bash
npx getdesign@latest add stripe.ai
```

## ⚡ Tips

1. **Drop** DESIGN.md di project root
2. **Beritahu** AI agent gunakan file tersebut
3. AI akan baca **exact tokens**: hex colors, fonts, spacing
4. Hasil lebih consistent & professional
