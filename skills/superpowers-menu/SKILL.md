# Superpowers Menu - Development Workflow

## Available Skills

| Command | Skill | Description |
|---------|-------|-------------|
| `/brainstorm` | brainstorming | Socratic design refinement - SEBELUM menulis code |
| `/plans` | writing-plans | Buat implementation plan detail (2-5 menit per task) |
| `/tdd` | test-driven-development | RED-GREEN-REFACTOR TDD cycle |
| `/debug` | systematic-debugging | 4-phase root cause debugging |
| `/subagent` | subagent-driven-development | Parallel subagent execution |
| `/codereview` | requesting-code-review | Pre-merge code review |
| `/finish` | finishing-a-development-branch | Merge/PR decision workflow |
| `/worktree` | using-git-worktrees | Parallel branch development |

## Cara Penggunaan

Ketik command di atas untuk load skill tertentu:

```
use skill tool to load superpowers/brainstorming
```

## Workflow Default Superpowers

```
Brainstorming → Writing Plans → TDD → Code Review → Finish
```

## Quick Reference

- **Mau mulai fitur baru?** → `/brainstorm`
- **Sudah ada spec, mau plan?** → `/plans`
- **Mau implement dengan TDD?** → `/tdd`
- **Ada bug ingin systematic debug?** → `/debug`
- **Mau execute plan dengan subagent?** → `/subagent`
- **Mau review code sebelum merge?** → `/codereview`
- **Task selesai, mau finish branch?** → `/finish`
