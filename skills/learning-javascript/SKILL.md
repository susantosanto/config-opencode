# Learning JavaScript — Dari Fundamental ke Mastery

## 📋 Overview
Skill ini adalah learning workflow komprehensif untuk JavaScript. Mulai dari **nol absolut** sampai **siap production**. Setiap tahap dibangun di atas tahap sebelumnya.

## 🗺️ Learning Path

### Fase 1: Fondasi (Minggu 1-2)
**Target:** Bisa menulis program JavaScript sederhana

| # | Topik | Target Penguasaan |
|---|-------|-------------------|
| 1 | **Apa itu JavaScript?** | Sejarah, cara jalan di browser vs Node.js, tools setup |
| 2 | **Variables** | `let`, `const`, `var`, tipe data (string, number, boolean, null, undefined) |
| 3 | **Operators** | Aritmatika, perbandingan, logika |
| 4 | **Conditionals** | `if/else`, `switch`, ternary operator |
| 5 | **Loops** | `for`, `while`, `do...while` |
| 6 | **Functions** | Deklarasi, expression, arrow functions, parameters, return |
| 7 | **Array** | Membuat, mengakses, method dasar (`push`, `pop`, `forEach`) |
| 8 | **Object** | Membuat, properti, method, nested object |

**🚀 Mini Project:** Kalkulator sederhana, Tebak angka

### Fase 2: Intermediate (Minggu 3-4)
**Target:** Bisa membuat aplikasi web interaktif sederhana

| # | Topik | Target Penguasaan |
|---|-------|-------------------|
| 1 | **DOM Manipulation** | `querySelector`, `addEventListener`, mengubah konten/style |
| 2 | **Events** | Click, submit, keydown, mouse events, event delegation |
| 3 | **Array Methods** | `map`, `filter`, `reduce`, `find`, `sort` |
| 4 | **Spread & Rest** | `...` operator, destructuring |
| 5 | **Callback & Higher Order Functions** | Function sebagai argument |
| 6 | **Error Handling** | `try/catch/finally`, custom errors |

**🚀 Mini Project:** Todo List, Shopping Cart

### Fase 3: Advanced (Minggu 5-6)
**Target:** Paham konsep fundamental JavaScript secara deep

| # | Topik | Target Penguasaan |
|---|-------|-------------------|
| 1 | **Async JavaScript** | Callback hell → Promises → `async/await` |
| 2 | **Fetch API & AJAX** | GET/POST request, handling response, error handling |
| 3 | **Closures** | Apa itu closure, kenapa penting, use cases |
| 4 | **`this` keyword** | Implicit binding, explicit binding, arrow function |
| 5 | **Prototypes & Classes** | Prototypal inheritance, ES6 classes |
| 6 | **Modules** | `import`/`export`, CommonJS vs ES modules |

**🚀 Mini Project:** Weather App dengan API

### Fase 4: Real-World Engineering (Minggu 7-8)
**Target:** Bisa membuat aplikasi production-ready

| # | Topik | Target Penguasaan |
|---|-------|-------------------|
| 1 | **Debugging** | Chrome DevTools, breakpoints, network tab |
| 2 | **Testing** | Unit test dengan Jest, TDD basic |
| 3 | **Performance** | Debounce/throttle, lazy loading, code splitting |
| 4 | **Security** | XSS prevention, input validation, CORS |
| 5 | **Async Patterns** | Promise.all, Promise.race, error boundaries |
| 6 | **State Management** | Pattern sederhana sebelum pake framework |

**🚀 Final Project:** Aplikasi CRUD lengkap dengan API

## 🧠 Problem Solving Workflow

Setiap kali menghadapi masalah coding, gunakan framework ini:

```
1. PAHAMI MASALAH
   - "Apa yang harus dilakukan program ini?"
   - Tulis dengan kata-kata sendiri
   
2. BREAKDOWN
   - "Langkah kecil apa saja yang diperlukan?"
   - Tulis pseudocode
   
3. IDENTIFIKASI POLA
   - "Pernah lihat pola seperti ini sebelumnya?"
   - "Konsep apa yang relevan?"
   
4. IMPLEMENTASI
   - Mulai dari langkah terkecil
   - Test setiap langkah
   
5. REFLEKSI
   - "Apa yang saya pelajari?"
   - "Bagaimana bisa lebih baik?"
```
