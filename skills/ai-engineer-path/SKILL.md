# AI Engineer Learning Path - Python & Langchain

## 📚 Roadmap

### Phase 1: Python Fundamentals (Week 1-2)
- [ ] Python basics: variables, functions, classes
- [ ] Data structures: list, dict, set, tuple
- [ ] File I/O and error handling
- [ ] Virtual environments and pip
- [ ] Type hints dan debugging

### Phase 2: AI/ML Basics (Week 3-4)
- [ ] NumPy basics
- [ ] Pandas for data manipulation
- [ ] Basic statistics concepts
- [ ] OpenAI API fundamentals
- [ ] Environment and API key management

### Phase 3: Langchain Core (Week 5-8)
- [ ] Langchain LLM wrappers
- [ ] Chains and LCEL
- [ ] Prompt templates
- [ ] Memory and conversation context
- [ ] Output parsers
- [ ] Vector stores and embeddings
- [ ] RAG (Retrieval-Augmented Generation)

### Phase 4: Advanced Langchain (Week 9-12)
- [ ] Agents and Tools
- [ ] Custom agents
- [ ] Multi-agent systems
- [ ] Langchain Smith / LangGraph
- [ ] Streaming and async
- [ ] Production deployment

### Phase 5: Specialization (Ongoing)
- [ ] Computer Vision with AI
- [ ] Audio/Speech AI
- [ ] Fine-tuning models
- [ ] MLOps fundamentals
