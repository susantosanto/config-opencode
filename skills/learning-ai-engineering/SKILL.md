# Learning AI Engineering

## 📋 Overview
Skill ini adalah learning workflow untuk menjadi **AI Engineer**. Dari fundamental Python, matematika dasar, sampai membangun dan mendeploy AI agents dan LLM applications.

## 🗺️ Learning Path

### Fase 0: Prerequisites (Minggu 1)
**Target:** Setup environment dan paham tools

| # | Topik | Target Penguasaan |
|---|-------|-------------------|
| 1 | **Python Refresher** | Functions, classes, file I/O, pip |
| 2 | **Jupyter Notebook** | Install, run cells, markdown, visualisasi |
| 3 | **Git for AI** | Version control untuk notebook dan model |
| 4 | **Command Line Basics** | Terminal, environment variables, PATH |

### Fase 1: Mathematical Foundation (Minggu 2-3)
**Target:** Paham matematika yang diperlukan untuk ML

| # | Topik | Target Penguasaan |
|---|-------|-------------------|
| 1 | **Linear Algebra** | Vectors, matrices, dot product, eigenvalues |
| 2 | **Calculus** | Derivatives, gradients, chain rule |
| 3 | **Statistics** | Mean, median, variance, standard deviation |
| 4 | **Probability** | Bayes theorem, distributions, MLE |
| 5 | **Optimization** | Gradient descent, loss functions |

**🔬 Praktik:** Implementasi linear regression dari scratch

### Fase 2: Data Science & ML (Minggu 4-6)
**Target:** Bisa menganalisis data dan bikin model ML

| # | Topik | Target Penguasaan |
|---|-------|-------------------|
| 1 | **NumPy** | Array operations, broadcasting, vectorization |
| 2 | **Pandas** | DataFrame, groupby, merge, pivot, datetime |
| 3 | **Matplotlib & Seaborn** | Visualisasi data, exploratory analysis |
| 4 | **Scikit-learn** | Train/test split, preprocessing, pipelines |
| 5 | **Supervised Learning** | Regression, classification, tree-based models |
| 6 | **Unsupervised Learning** | Clustering, PCA, dimensionality reduction |
| 7 | **Model Evaluation** | Cross-validation, metrics, confusion matrix |
| 8 | **Feature Engineering** | Encoding, scaling, feature selection |

**🚀 Mini Project:** Prediksi harga rumah, Klasifikasi email spam

### Fase 3: Deep Learning (Minggu 7-8)
**Target:** Paham neural networks dan bisa training model

| # | Topik | Target Penguasaan |
|---|-------|-------------------|
| 1 | **Neural Networks Basics** | Perceptron, activation functions, backpropagation |
| 2 | **PyTorch/TensorFlow** | Tensors, autograd, model building |
| 3 | **CNN** | Convolution, pooling, image classification |
| 4 | **RNN & LSTM** | Sequence data, text classification |
| 5 | **Transfer Learning** | Pre-trained models, fine-tuning |
| 6 | **Training Tips** | Learning rate, batch size, regularization, dropout |

**🚀 Mini Project:** Image classifier, Text sentiment analysis

### Fase 4: LLM & NLP (Minggu 9-10)
**Target:** Bisa menggunakan dan fine-tune LLM

| # | Topik | Target Penguasaan |
|---|-------|-------------------|
| 1 | **NLP Fundamentals** | Tokenization, embeddings, transformers |
| 2 | **Transformer Architecture** | Attention, self-attention, multi-head attention |
| 3 | **LLM APIs** | OpenAI API, Claude API, prompt engineering |
| 4 | **RAG (Retrieval Augmented Generation)** | Vector databases, embeddings, chunking |
| 5 | **LangChain/LlamaIndex** | Chains, agents, tools, memory |
| 6 | **Fine-tuning** | LoRA, QLoRA, dataset preparation |
| 7 | **Evaluation** | LLM-as-judge, BLEU, ROUGE, human eval |

**🚀 Mini Project:** Chatbot dengan RAG, Document Q&A system

### Fase 5: AI Agents & Production (Minggu 11-12)
**Target:** Bisa membangun dan mendeploy AI agents

| # | Topik | Target Penguasaan |
|---|-------|-------------------|
| 1 | **AI Agents Concepts** | Reasoning, planning, tool use, memory |
| 2 | **Agent Frameworks** | LangGraph, CrewAI, AutoGen |
| 3 | **MCP Protocol** | Model Context Protocol, tool servers |
| 4 | **API Development** | FastAPI untuk serving model/agent |
| 5 | **Vector Databases** | Pinecone, Weaviate, Chroma, Qdrant |
| 6 | **Deployment** | Docker, cloud deployment (Railway, HuggingFace) |
| 7 | **Monitoring** | Trace/Arize, logging, cost tracking |

**🚀 Final Project:** AI Agent untuk otomatisasi task tertentu

## 🧠 AI Engineering Mindset

```
SETIAP PROBLEM, TANYAKAN:

1. "APAKAH ML DIPERLUKAN?"
   → Atau cukup rule-based?
   
2. "DATA APA YANG TERSEDIA?"
   → Quantity, quality, label?
   
3. "METRIK KEBERHASILAN?"
   → Accuracy? Latency? Cost?
   
4. "TRADE-OFFS?"
   → Model besar vs kecil? Akurasi vs kecepatan?
   
5. "PRODUCTION READY?"
   → Monitoring? Retraining? Bias?
```
