# Learning Python — Dari Fundamental ke Mastery

## 📋 Overview
Skill ini adalah learning workflow komprehensif untuk Python. Mulai dari **nol absolut** sampai **siap untuk AI/ML**.

## 🗺️ Learning Path

### Fase 1: Fondasi (Minggu 1-2)
**Target:** Bisa menulis script Python sederhana

| # | Topik | Target Penguasaan |
|---|-------|-------------------|
| 1 | **Apa itu Python?** | Install, REPL, run script, Python philosophy |
| 2 | **Variables & Types** | `int`, `float`, `str`, `bool`, `None`, type() |
| 3 | **Strings** | Concatenation, f-strings, methods, slicing |
| 4 | **Lists & Tuples** | Membuat, mengakses, slicing, methods |
| 5 | **Dictionaries & Sets** | Key-value, methods, use cases |
| 6 | **Operators** | Aritmatika, perbandingan, logika, identity, membership |
| 7 | **Conditionals** | `if/elif/else`, truthy/falsy |
| 8 | **Loops** | `for`, `while`, `range()`, `enumerate()` |
| 9 | **Functions** | `def`, parameters, return, docstrings, scope |

**🚀 Mini Project:** Tebak angka, Konversi suhu

### Fase 2: Intermediate (Minggu 3-4)
**Target:** Bisa membuat program dengan multiple files

| # | Topik | Target Penguasaan |
|---|-------|-------------------|
| 1 | **List Comprehension** | Syntax, nested comprehension |
| 2 | **Lambda Functions** | Anonymous functions, `map()`, `filter()` |
| 3 | **File I/O** | Read/write files, context manager (`with`) |
| 4 | **Error Handling** | `try/except/else/finally`, raise exceptions |
| 5 | **Modules & Packages** | `import`, `__name__ == '__main__'`, pip |
| 6 | **Working with Libraries** | `datetime`, `json`, `os`, `sys`, `re` |

**🚀 Mini Project:** To-Do List CLI, File organizer

### Fase 3: Advanced Python (Minggu 5-6)
**Target:** Paham OOP dan bisa struktur kode yang baik

| # | Topik | Target Penguasaan |
|---|-------|-------------------|
| 1 | **OOP Dasar** | Class, object, `__init__`, methods vs functions |
| 2 | **OOP Lanjutan** | Inheritance, polymorphism, encapsulation |
| 3 | **Magic Methods** | `__str__`, `__repr__`, `__len__`, `__eq__` |
| 4 | **Decorators** | `@property`, `@staticmethod`, custom decorators |
| 5 | **Generators & Iterators** | `yield`, generator expressions |
| 6 | **Context Managers** | Custom context manager, `contextlib` |

**🚀 Mini Project:** Bank Account system, Library management

### Fase 4: Data & ML Path (Minggu 7-8+)
**Target:** Siap untuk数据分析 dan Machine Learning

| # | Topik | Target Penguasaan |
|---|-------|-------------------|
| 1 | **NumPy** | Arrays, broadcasting, vectorization |
| 2 | **Pandas** | DataFrame, Series, groupby, merge |
| 3 | **Matplotlib/Seaborn** | Line plot, bar chart, histogram, scatter plot |
| 4 | **Scikit-learn Dasar** | Train/test split, fit/predict, metrics |
| 5 | **Supervised Learning** | Linear regression, decision tree, SVM |
| 6 | **Unsupervised Learning** | K-means, PCA |

**🚀 Mini Project:** Data analysis dashboard, Simple ML predictor

## 🧠 Problem Solving Framework untuk Python

```
1. ANALISIS MASALAH
   "Apa input dan output yang diharapkan?"

2. PSEUDOCODE
   "Tulis langkah-langkah dalam bahasa Indonesia dulu"

3. PYTHONIC APPROACH
   "Adakah cara Python yang lebih idiomatic?"
   → List comprehension? Built-in functions?

4. IMPLEMENTASI
   Mulai dari fungsi terkecil, test setiap fungsi

5. REFLEKSI
   "Bagaimana cara meningkatkan readability?"
   "Apakah sudah handling edge cases?"
```
