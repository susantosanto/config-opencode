## 🤖 TELEGRAM BOT INTEGRATION

### Akses Remote via Telegram
User memiliki Telegram Bot **@laptopXcodeBot** yang memungkinkan akses remote ke laptop:

**Fitur Telegram Bot:**
1. **Remote Coding** - Kirim perintah coding dari Telegram
2. **File Operations** - Akses, baca, tulis, modifikasi file di laptop
3. **Session Management** - Kelola sesi OpenCode dari HP
4. **Project Switching** - Switch antar project dari Telegram
5. **System Control** - Start/stop OpenCode server dan bot

**Command Telegram:**
- `/status` - Lihat status server dan project
- `/projects` - List semua project
- `/new` - Buat sesi baru
- `/sessions` - List dan switch sesi
- `/abort` - Batalkan task yang berjalan

**Akses File via Telegram:**
User bisa meminta untuk:
- Membaca file tertentu (langsung via prompt)
- Menulis/memodifikasi file
- Melihat struktur direktori
- Membuat file baru

**Contoh penggunaan:**
- "Baca file C:\Users\USER\Documents\data.xlsx"
- "Buat file baru di D:\Project\test.js dengan isi: ..."
- "List semua file di D:\Programming"
